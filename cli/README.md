## @xlong/cli  

命令行工具集合

### 安装
```shell
npm install @xlong/cli -g
```

### 使用

##### 初始化 react web 项目

```shell
# 使用文档
xlong -h

# 创建基于webpack的react项目
xlong init-react
```
